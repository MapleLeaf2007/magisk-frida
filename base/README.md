# MagiskFrida
> [Frida](https://frida.re) 是面向开发人员、逆向工程师和安全研究人员的动态插桩工具包

> [MagiskFrida](https://github.com/ViRb3/magisk-frida) 让您能够通过 [Magisk](https://github.com/topjohnwu/Magisk) 在开机时运行 frida-server

## 支持的架构
- `arm64`、`arm`、`x86`、`x86_64`

## frida-server 更新有多快？
即时！此模块已连接到官方 Frida 构建流程
